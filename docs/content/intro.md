# はじめに

このサイトは設計ドキュメントを管理するためのドキュメントサイトです。

## ドキュメントの種類

- **ADR (Architecture Decision Records)**: アーキテクチャの意思決定を記録
- **アーキテクチャ**: システム全体の構造と設計
- **仕様書**: 機能やAPIの詳細仕様
- **ガイド**: 開発者向けの手順書やベストプラクティス
- **Frontend**: フロントエンド（React/Next.js）の設計ドキュメント
- **BFF**: Backend for Frontend（Node.js）の設計ドキュメント
