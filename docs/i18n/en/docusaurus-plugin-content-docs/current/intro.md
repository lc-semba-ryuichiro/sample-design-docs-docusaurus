# Introduction

This site is a documentation site for managing design documents.

## Document Types

- **ADR (Architecture Decision Records)**: Recording architectural decisions
- **Architecture**: Overall system structure and design
- **Specifications**: Detailed specifications for features and APIs
- **Guides**: Developer guides and best practices
- **Frontend**: Frontend (React/Next.js) design documentation
- **BFF**: Backend for Frontend (Node.js) design documentation
